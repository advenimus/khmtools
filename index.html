Start Zoom Tool Help

Purpose

This tool allows you to quickly launch Zoom meetings with one click. It can automatically join your congregation's recurring meeting using the configured Meeting ID.

How It Works

  1. Click the "Launch Zoom" button to start Zoom
  2. If a Meeting ID is configured, Zoom will automatically join that meeting
  3. If you're the host and signed in to Zoom, the meeting will start automatically
  4. If no Meeting ID is set, Zoom will just open normally

Requirements

  • Zoom Desktop Client: Must be installed on your computer
  • Meeting ID: Configure in Settings (⚙️) under Meeting Settings
  • Host Auto-Start: You must be signed in to Zoom with host privileges

Settings

Configure these in the main Settings (⚙️ button):

  • Meeting ID: Your congregation's recurring Zoom meeting ID
  • Zoom Application: Path to Zoom if not found automatically

Important Notes

⚠️ Zoom Auto-Launch Requirements:

  • You must be signed into Zoom for automatic meeting launch to work
  • Configure your Meeting ID in Settings → Meeting Settings
  • If you're the host, Zoom will automatically start the meeting
  • If you're not the host, you'll join the waiting room

Tips

  • For hosts: Sign in to Zoom before the meeting for auto-start capability
  • For attendants: This tool provides quick access to join the meeting
  • Use with the "Default Tool" setting to open this automatically when KHM Tools starts
  • The Zoom path is shared across all tools that use Zoom

Media Launcher Help

How Media Launcher Works

The Media Launcher tool helps you start all your meeting media applications in the correct order with a single click. It automatically launches each application with a short delay between them to ensure everything starts properly.

Launch Sequence

  1. Custom Message (optional) - Displays a welcome message for attendees
  2. OBS Studio - Starts with virtual camera enabled for video sharing
  3. Meeting Media Manager - Opens your meeting media presentation software
  4. Zoom - Launches and joins your configured meeting

Settings Explained

Tool Toggles

In Settings → Media Launcher, you can enable or disable individual tools:

  • Toggle each tool on/off based on your needs
  • Disabled tools will be skipped in the launch sequence

Custom Message Settings

  • Display When:
    • Always - Shows every time you launch
    • Weekend meetings only - Shows only on your configured weekend meeting day
    • Never - Disables the custom message
  • Message Title: The heading shown in the popup
  • Message Text: Your custom welcome message (supports multiple lines)
  • Display Time: How long the message shows (in seconds)

Application Paths

Configure paths in Settings → Application Paths:

  • Default paths are automatically detected for most applications
  • Use the Browse button to manually select application locations
  • Ensure all paths point to valid application files

Important Notes

⚠️ Zoom Auto-Launch Requirements:

  • You must be signed into Zoom for automatic meeting launch to work
  • Configure your Meeting ID in Settings → Meeting Settings
  • If you're the host, Zoom will automatically start the meeting
  • If you're not the host, you'll join the waiting room

Troubleshooting

  • Application not launching: Verify the application path in Settings
  • Zoom not joining meeting: Ensure you're signed in and have a valid Meeting ID
  • Custom message not showing: Check your display settings and meeting schedule
